Prototype Fields
  • pattern:The pattern string

Static Methods
  • new:func(pattern)
    Creates a new regex
    pattern The pattern to compile
    The regex is compiled when this method is called. The alternate /pattern// syntax may be used; in that case the regex is compiled during compilation of the rest of the program.
Prototype Methods
  • matches:func(string, [strict])
    Checks for matches in a given string
    string The string against which to match
    [strict] Should it be a strict match?
    Returns yes or no depending on whether or not the string matches the regex. If strict is set to yes, the string must me matched in its entirety. strict defaults to no.
  • search:func(string)
    Returns the index of the first occurrence of the regex in a given string
    string The string to scan
    Scans through the string (automatically not strict) and returns the index of the first occurrence or -1 if the pattern is not found.
