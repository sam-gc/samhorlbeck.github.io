Lanky Standard Library
Time
A standard (not yet fully implemented) way to deal with times.
Back
Prototype Fields
  • second:The current second value
  • minute:The current minute value
  • hour:The current hour value, 0-23
  • month:The current month of the year, 0-11
  • dayOfWeek:The current day of the week, 0-6
  • dayOfYear:The current day of the year, 0-364
  • year:The current year
Static Methods
  • new:func([table])
    Standard initializer for time objects
    [table] Optional initial values
    If no argument is supplied, a new time object is returned as the current time and date. Otherwise, it is initialized based on the keys in table. The names of the keys should be the same as the names of the Prototype Fields above.
  • unix:func()
    Returns the number of milliseconds since the epoch
Prototype Methods
  • format:func(format)
    Formats the time object according to a format string
    format The format string
    The format string should be useable by the C standard strftime function.
  • stringify_:func()
    Custom stringification function
