Lanky Standard Library
Math
A common class to aggregate mathematical functions. All of the static methods below can be used standalone (i.e. they are not bound to the Math class).

  • pi:A floating-point constant representing π

Static Methods
  • rand:func()
    Simple wrapper for C's rand function
    The random number generator is initialized when the interpreter spins up.
  • randInt:func(min, max)
    Generates a random number between two bounds
    min The minimum value (inclusive) for the generator
    max The maximum value (exclusive)
    This function will return a random number between min and max. Mathematically the range of this function can be expressed as [min, max).
  • shuffle:func(arr)
    Shuffles the given array
    arr The array to shuffle
    This method will return an array that contains the shuffled elements of the input.
  • range:func(max)
    Returns an array of a range of numbers
    max The max value in the range (exclusive)
    This method will return an array containing the range of numbers [0, max).
  • atan2:func(y, x)
    Wrapper around C's atan2 function
    y The arctangent y value
    x The arctangent x value
